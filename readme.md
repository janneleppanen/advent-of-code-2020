# Advent of Code 2020

## Running a task

Run all daily tasks:  
`npm start`  
or for specific day:  
`npm start 1`  
`npm start 2`  
and so on...

## Run test

Run all tests:  
`npm test`  
or for specific day:  
`npm test 1`  
`npm test 2`  
and so on...
